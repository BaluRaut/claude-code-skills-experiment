# DOC-3 · Doctors directory (seed + list)

Type: Story · Points: 3 · Priority: High

## Description
As a patient, I want to see the available doctors so I can choose one to book
with.

## Acceptance criteria
- AC-1 On first run, the app is seeded with at least 4 doctors (varied
  specialties) if none exist yet.
- AC-2 A Doctors screen lists each doctor's name and specialty.
- AC-3 The list of doctors is available to the booking form (DOC-4) as
  selectable options.
- AC-4 Seeding runs once — reloading does not duplicate doctors.

## Out of scope
- Adding/editing/deleting doctors from the UI.
